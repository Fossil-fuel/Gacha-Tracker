About

Gacha Tracker is a browser-based app for tracking daily and weekly tasks across multiple gacha games. It's fully manual — no game APIs or accounts — and stores data locally in your browser.

Nothing here is automatic or connected to game accounts. Everything is manual on purpose so it stays safe, flexible, and ban‑free.

Core Features

  • Home — Overview of today's progress with progress bars for Dailies, Weeklies, and Endgame, plus a checklist ordered by due date.
  • Dailies — Track daily tasks per game, with server reset times and optional currency tracking.
  • Weeklies — Track weekly content with configurable reset schedules.
  • Endgame — Track endgame modes with partial credit and currency earned per completion.
  • Extracurricular — Custom tasks for events, story, or other reminders with start/end dates.
  • Attendance — Weekly participation view and history with pie charts.
  • Data — Breakdown of currency earned across games and task types.
  • Games — Add, edit, or remove games. Start from presets or build custom.

Other Features

  • Server regions (America, Asia, Europe) and DST handling for reset times
  • Custom themes (presets and custom colors)
  • Export/import data as JSON
  • Optional 30-day simulation for testing
  • List and grid views for tasks

How Time Works

Reset times and dates are based on your chosen timezone and each game's server region. Here's what you need to know:

  • Display timezone — In Settings → Time, choose your timezone. This controls the date and time shown in the sidebar, when the calendar shows "today," and how reset times are displayed. You can use your device's local time or pick a fixed timezone.
  • Server regions — Each game has a server (America, Asia, or Europe). Reset times follow that server's clock. For example, if a game resets at 4:00 AM server time, it resets at 4:00 AM in that region.
  • Daylight Saving Time — For America and Europe servers, the tracker can adjust reset times when DST changes (e.g. 4:00 AM during DST, 3:00 AM when DST ends). Asia has no DST. You can turn this on or off per task or in settings.
  • What counts as "today" — A daily period runs from one reset to the next. So if dailies reset at 3:00 AM, then 2:59 AM still belongs to yesterday's tasks; 3:00 AM starts today's.
  • Weeklies — Each weekly task has a start day (e.g. Monday) and time. The week runs from that moment for 7 days. The "time remaining" shows how long until the current week ends.
  • Endgame — Endgame tasks use cycles (e.g. every 2 weeks). Each cycle has a deadline. Time remaining is until the end of the current cycle.

The sidebar shows your current time, the next reset countdown, and a comparison of reset times with and without DST. Check Settings → Time to sync your timezone or change how dates and times are displayed.
